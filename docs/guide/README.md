## 申请流程
1. 添加QQ2121349115好友
2. 私聊发送".申请机哥"
3. 点击链接填写申请表
4. 私聊发送".查看申请"，查看情况
5. 等着排队
6. 管理员收到 申请
7. 管理员正在处理
8. 管理员处理完成
