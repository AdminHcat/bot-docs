# 介绍

HcatSuAiBot 是一款Tencent QQ机器人，可在群内按需智能应答指令



## 申请流程

1. 添加QQ2861878208好友
2. 邀请QQ2861878208进群
3. 私聊发送".申请SuAiBot"
4. 点击链接填写申请表
5. 私聊发送".查看申请"，查看情况
6. 耐心等候排队
7. 管理员收到申请
8. 管理员开始处理
9. 管理员处理完成

- 验证10分钟过期，只能自行使用用，切勿发与别人填写
- QQ2861878208仅用于处理申请，不回复任何群聊信息
