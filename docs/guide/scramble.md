# 打乱

## 介绍

本功能可以根据发出的指令给出魔方的打乱公式，方便群内多人PK。

## 指令魔方对应表

| 指令 | 魔方种类 |
|---|---|
| .2 | 2阶魔方 |
| .3 | 3阶魔方 |
| .4 | 4阶魔方 |
| .5 | 5阶魔方 |
| .py | 金字塔魔方 |
| .sk | 斜转魔方 |
| .cl | 魔表魔方 |
| .sq | SQ1魔方 |
